Image API Documentation

POST /upload

Description: Upload an image (JPEG, PNG, GIF)

Headers:

Authorization: Bearer <token>

Body (multipart/form-data):

file=<your image file>

Success Response:

{
  "message": "Upload successful",
  "id": "abc123",
  "url": "https://cdn.example.com/abc123/original"
}

GET /image/:id

Description: Download the original image by ID

Response: Raw image file

DELETE /image/:id

Description: Delete an image by ID

Headers:

Authorization: Bearer <token>

Success Response:

{
  "message": "Image deleted",
  "id": "abc123"
}

GET /metrics/uploads

Description: Get upload metrics

Headers:

Authorization: Bearer <token>

Success Response:

[
  {
    "id": "abc123",
    "size": 12345,
    "uploadedBy": "user@example.com",
    "type": "image/png",
    "timestamp": 1700000000000
  }
]
